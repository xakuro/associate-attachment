Associate the media library image with the post.

== Description ==

Associate images and posts in the media library at once (set "Uploaded to"). 

= Supported media files =

* Images in post content
* Featured images
* WooCommerce products and product gallery images
* Image Field in Advanced Custom Fields plugin
* Image Field in Smart Custom Fields plugin

== Installation ==

1. Upload the `associate-attachment` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
